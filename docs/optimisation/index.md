# 优化

优化问题一直以来都是一个很重要的问题。令人遗憾的是，我们现有的知识对于非凸优化问题还处于儿童水准--大多数求解非凸优化问题的方法也只是利用凸优化的思路去求解，或者直接将其松弛成为一个凸优化问题。

在本章，我们首先介绍什么是凸、什么是凸函数，随后我们将介绍凸优化的方法。
