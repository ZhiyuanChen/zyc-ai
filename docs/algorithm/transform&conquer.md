# 变治法(Transform and Conquer))

顾名思义，变治法由两个步骤构成。

+ 变换问题
+ 解决变换后的问题

理所当然的，变换后的问题应该更容易被解决。那，有哪些变换呢？

+ 实例化简（Instance simplification）
+ 表现改变（Representation Change）
+ 问题化简（Problem reduction）
